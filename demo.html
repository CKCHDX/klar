Features of This Render Engine

  • Custom HTML Parser - Parses HTML into a DOM tree
  • CSS Style Computation - Applies styles to elements
  • Rendering Pipeline - Paints content to viewport
  • No WebEngine - Built completely from scratch

Technical Details

The render engine uses:

  • BeautifulSoup for HTML parsing
  • PyQt6 for GUI and painting
  • Custom layout and rendering logic
  • Requests library for HTTP
